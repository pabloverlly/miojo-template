# Miojo Template :ramen:
Aplicação web base em Node.js


![alt text](https://i.imgur.com/1Ly5HyE.png)


### instação e uso

 1. ````git clone https://github.com/pabloverlly/miojo-template.git````
 2. ````cd miojo-template````
 3. ````unzip assets.zip````
 4. ````npm install````
 5. ```` npm start ````
